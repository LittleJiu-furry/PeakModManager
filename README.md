
## 这是什么
这是一个专门对于《PEAK》这个游戏制作的模组管理器。

## 如何使用
- 在[Releases]()页面下载最新版本的模组管理器。
- 解压下载的文件，并运行其中的`main.exe`。

## 如何构建
如果你想从源码构建这个项目，请参考下面的操作：
1. 克隆这个仓库到你的本地。
2. 确保你已经安装了Python 3.12和pnpm.
3. 在ui目录下运行`pnpm install`安装前端依赖，在项目根目录下运行`pip install -r requirements.txt`安装后端依赖。
4. 使用`nuitka`构建项目。


## 如何贡献
如果你有兴趣为这个项目贡献代码，请参考下面的步骤：
1. Fork这个仓库到你的账户。
2. 克隆你的仓库到本地。
3. 创建一个`feat\<feature-name>`分支，并在该分支上进行开发。
4. 完成后，请按照[源码构建](#如何构建)方法构建项目，测试你的更改。
5. 提交PR。
